A high-end, interactive photography portfolio website built with modern web technologies. This project showcases a seamless blend of elegant design, smooth animations, and responsive layouts, tailored for creative professionals.

## ‚ú® Features

- **Seamless Navigation** ‚Äì Transparent navbar with `mix-blend-mode: difference` and GSAP-powered hover animations.
- **Hero Section** ‚Äì Full-screen background with a grayscale/color toggle button (B&W ‚Üî Color).
- **Infinite Marquee** ‚Äì Two rows of images scrolling in opposite directions (left & right) for dynamic visual appeal.
- **Card Stack Shuffle** ‚Äì Interactive project cards that shuffle with smooth GSAP animations. Click "View Detail" to open a modal with project info.
- **Dual-Scroll Parallax** ‚Äì A custom section with two columns moving in opposite directions as you scroll, creating a cinematic storytelling effect.
- **About Section** ‚Äì Clean layout with experience counters that animate when scrolled into view.
- **Contact Form** ‚Äì Functional form (demo mode) integrated with FormSubmit, plus a grayscale map.
- **Mobile Menu** ‚Äì Elegant slide-in menu with social links and staggered animations.
- **Smooth Scrolling** ‚Äì Powered by Lenis for a butter-smooth experience.
- **Fully Responsive** ‚Äì Optimized for all devices, from iPhone SE to 4K displays.
